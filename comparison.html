05

Internationalization & Routing

The old site toggles languages with CSS display:none. The new site uses proper URL-based routing.

Old Site

Query Parameter Routing

https://draestelamas.com/?lang=ca
https://draestelamas.com/?lang=es

All 3 languages loaded in HTML, hidden via CSS. Sharing a link does not preserve language. Search engines see duplicate content.

→
New Site

URL Path Routing

https://draestelamas.com/
https://draestelamas.com/es/
https://draestelamas.com/en/

Each language has its own URL. Sharing a link preserves language. Search engines index each language independently with proper hreflang.
